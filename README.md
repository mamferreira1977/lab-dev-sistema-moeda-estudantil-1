## lab-dev-sistema-aluguel-carros
### Pretende-se desenvolver um sistema para apoio à gestão de aluguéis de automóveis que permita efetuar, cancelar e modificar pedidos através da Internet.
#### Modelagem do Sistema
---

<strong>1-Histórias de Usuário:<strong>
#### Cliente
#### Como cliente, quero me cadastrar no sistema para poder solicitar aluguel de automóveis.
#### Como cliente, quero fazer login para acessar minha área restrita.
#### Como cliente, quero introduzir um pedido de aluguel para solicitar um automóvel.
#### Como cliente, quero consultar meus pedidos para acompanhar suas informações.
#### Como cliente, quero modificar um pedido para corrigir ou atualizar dados antes da contratação.
#### Como cliente, quero cancelar um pedido para desistir da solicitação.
#### Como cliente, quero consultar o status do pedido para saber se foi aprovado, reprovado ou está em análise.

#### Agente
#### Como agente, quero autenticar-me no sistema para acessar os pedidos disponíveis.
#### Como agente, quero consultar os pedidos cadastrados para analisá-los.
#### Como agente, quero modificar pedidos quando necessário para adequação contratual ou documental.
#### Como agente, quero avaliar financeiramente um pedido para decidir sobre sua aprovação.
#### Como agente, quero aprovar um pedido quando o parecer financeiro for favorável.
#### Como agente, quero reprovar um pedido quando o parecer financeiro for desfavorável.
#### Como agente, quero executar o contrato após aprovação do pedido.

#### Banco
#### Como banco agente, quero registrar um contrato de crédito para os pedidos aprovados que envolvam financiamento.
